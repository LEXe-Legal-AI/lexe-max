# LEXE-MAX

Legal Tools API for LEXE Platform.

## Features

- Normattiva integration
- EUR-Lex integration  
- Knowledge Base (massimari giurisprudenziali)
- BM25 + Vector search
- Apache AGE graph queries

## Stack

- FastAPI
- PostgreSQL 17 + pgvector + Apache AGE + ParadeDB
- LiteLLM gateway

## Run

```bash
docker compose up -d lexe-max
```
